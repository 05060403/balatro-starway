# Balatro Starway
Vanilla friendly resource pack mod, replacing and renaming many jokers, adding a new custom deck, renaming and retexturing the seals, as well as renaming some of the cards and hands.


## Installation

[Steamodded] (https://github.com/Steamodded/smods) and [Malverk](https://github.com/Eremel/Malverk) are required.
Please make sure to install both before trying to install this mod.

1. Download the latest release from the [Releases](https://github.com/05060403/balatro-starway/releases) page or click the green Code button and `Download ZIP`.
2. Extract the `.zip` file to your Balatro `Mods` folder.
3. Make sure to activate the texture pack in the Malverk options.
